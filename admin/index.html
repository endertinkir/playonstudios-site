Private Studio Dashboard

One place for your iOS and Android game numbers.

Track store metrics, revenue, spend, DAU and release notes under /admin. This panel is wired for Firebase Auth + Firestore so you can keep the site static and still have a secure private backend.

Current model

Manual entry from this dashboard into Firestore. If your games already write data into Firebase, send the exact path/schema and it can be mapped into this panel.

Firebase setup is still using placeholders.

Open /admin/firebase-config.js, add your Firebase web config, and replace the admin email. Firestore security rules are included in /firebase/firestore.rules.

Setup Checklist

What this needs from you

  1. Firebase web config values from your project settings.
  2. The single admin email address you want to allow.
  3. Firebase Auth enabled with the Email/Password provider.
  4. Firestore database created and rules deployed from this repo.
  5. Optional: the existing Firebase collection/path if game data already lives there.
